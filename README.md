# Medical Record System Backend

## Overview

This project implements the backend logic for a medical record system. It includes patient records and notes, with behaviour validated through automated tests.

The system is designed as a backend library, not an interactive application.

---

## Requirements
- Python
- pytest

---

## Usage
This project does not accept user input. 
All functionality is executed by directly calling functions within automated tests via test suite in /tests

From root directory:

pytest
